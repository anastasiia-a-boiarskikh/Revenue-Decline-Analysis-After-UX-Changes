Анализ снижения выручки после UX-изменений. Кодовое название игры — Mines&Moles

- Сфера: game-аналитика
- Цель: сформулировать гипотезы о причинах снижения выручки и проанализировать изменения метрик, работая с деревом метрик продукта
- Результат: аналитический отчет с детальной информацией по метрикам, выводами и рекомендациями
- Стек: Python (pandas, matplotlib), JupyterLab
- Артефакты: дерево метрик, аналитический отчет, JupyterNotebook с расчетами и визуализациями

В отчете собраны визуализации динамики различных метрик продукта (по дереву метрик) с промежуточными выводами; общие выводы по проекту и рекомендации
